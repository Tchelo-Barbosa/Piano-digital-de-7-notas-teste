# 🎹  Mini $\color{red}{Piano{}}$ de 7 Notas com Arduino
Este é um projeto simples e interativo de um piano eletrônico desenvolvido com **Arduino Uno**. O projeto utiliza botões para simular as teclas de um piano e um buzzer (alto-falante piezoelétrico) para emitir as 7 notas musicais da escala diatônica (Dó, Ré, Mi, Fá, Sol, Lá, Si). 


---

##  $\color{red}{Funcionalidades{}}$

* Reprodução das 7 notas musicais básicas.
* Resposta imediata ao pressionar os botões.
* Totalmente compatível com simulação no Tinkercad.
